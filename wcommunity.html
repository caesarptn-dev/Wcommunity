🔧 Set Up Your Database

Wcommunity needs a free Firebase project so posts save online for everyone. Takes about 5 minutes.

1

Create a free Firebase project

Go to console.firebase.google.com → Click Add project → Follow the wizard (disable Analytics is fine).

2

Add a Web app & copy the config

Inside your project → Project Settings ⚙️ → Your apps → click the </> icon → register app → copy the firebaseConfig values shown.

3

Enable Realtime Database

Left sidebar → Build → Realtime Database → Create database → choose a region → start in test mode (you can add rules later).

4

Enable Email/Password Auth

Left sidebar → Build → Authentication → Get started → Email/Password → Enable → Save.

5

Paste your config below and save

Your config values are stored in this browser only. No one else sees them.

Paste Firebase Config

Config is saved in localStorage. To reset, open browser console and run localStorage.clear()
